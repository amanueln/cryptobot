Mission · momentum bot
Rotate into strong momentum coins, ride them without giving back peak, and exit before the thesis dies.
This roadmap covers only the acceleration / momentum rotation engine. Early-scanner Discord alerts, grid bot, and unrelated dashboard features are tracked elsewhere.
